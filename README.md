Ecommerce Insumos Médicos 

El propósito de esta actividad es desarrollar el frontend de un Ecommerce poniendo en práctica lo aprendido durante el trascurso del módulo. 

Funcionalidades
- Catálogo Dinámico: Generación de tarjetas de productos.
- Filtros: Clasificación de productos (Insumos, Instrumental, Vías Respiratorias, etc.)
- Carrito de Compras: LocalStorage
- Diseño Responsivo: Interfaz adaptada para dispositivos móviles y de escritorio.

Tecnología Utilizada
- HTML5 (Estructura)
- CSS3 (Estilos)
- Bootstrap (Diseño)
- JavaScript (Lógica y manejo del DOM)
- jQuery (Manipulación de eventos)
- Git-GitHub

Estructura Ecommerce
- index.html: Página principal del catálogo.
- detalle.html: Vista detallada de cada producto.
- carrito.html: Resumen de compras.
- assets/css/: Archivo de estilo.
- assets/js/: Lógica de la aplicación.
- assets/img/: Imágenes de los insumos.
